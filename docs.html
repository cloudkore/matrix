Core Features: Security and Convenience at Your Fingertips

NinjaBase offers a suite of integrated tools to help you organize and protect your sensitive information:

Secure Notes

You can securely save and retrieve your personal notes within NinjaBase. All notes are encrypted using advanced cryptographic methods before being stored, ensuring that your private thoughts and information remain confidential and accessible only by you.

Encrypted Password Manager

Never forget a password again! NinjaBase includes a robust password manager that allows you to store service names, usernames, and passwords for all your online accounts. Each password entry is encrypted individually, providing a secure vault for your credentials.

Cross-Platform Ephemeral File Storage

NinjaBase provides a convenient way to upload and download small files, making it a truly cross-platform solution for your file management needs. Whether you're on a desktop or mobile device, you can securely store and retrieve important documents or images with ease. Files are limited to a maximum size of 5MB per upload, ensuring efficient storage and transfer.

Security at its Core

NinjaBase is built with a strong emphasis on security, ensuring your data is protected:

  • Master Password Encryption: All sensitive data, including your notes and passwords, are encrypted using a strong master password that you set. This master password derives a unique encryption key using PBKDF2 with 200,000 iterations and SHA256 hashing, a highly secure key derivation function, ensuring your data is exceptionally well-protected. This key is session-based, meaning it's only active while you're logged in and interacting with your encrypted data.
  • Data Integrity: Your data is stored securely, and access to sensitive features like notes, password manager, and file storage requires verification with your master password.
  • Secure Authentication: User accounts and profile information (username and avatar) are managed through a robust authentication system.
